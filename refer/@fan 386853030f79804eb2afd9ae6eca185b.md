[ 영수 생각 ]

타겟 : 기존 플랫폼들이 유튜버, BJ, 스트리머, 대형 크리에이터 위주라면
@Fan은 아이돌 연습생, 인디 밴드, 대학생 크리에이터, 모델, 운동선수, 사진작가, 일러스트레이터, 메이크업 아티스트 등 수많은 창작자를 대상으로 함.

1. 아이돌에 국한되는게 아닌, 모든 ‘활동을 하는 사람들’(어떠한 행위를 하는 *운동이던, 예술이던 등등…)
2. 후원자가 후원을 하면 크리에이터는 그 후원에 대한 대가성 보상이 강제 되지 않아야함.

[ 영수 아이디어 ]

1. 이 사업이 지속 가능한 모델이 되려면 후원자에게 리워드를 부여해야함 
- **매일 오는 이유** → 퀘스트, 투표, 커뮤니티, 성장 기록
- **계속 돈을 쓰는 이유** → 한정 경험, 팬 영향력, 시즌 콘텐츠
- **떠나지 않는 이유** → 팬의 역사가 플랫폼에 축적됨 (팬 프로필에 후원 등급, 몇번째 팬, ㅇ)
1. 초기 MVP는 단순히 후원적인 요소와 메인 리워드들 만을 도입 → 빠르게 서비스 후 피드백 확인
점점 확장하면서 커뮤니티 기능, ㅇ

- 수현 아이디어
    
    앱 서비스 개요 : 후원을 통한 인플루언서와의 1:1 채팅  (초기 아이돌군에 초점을 두고 사용자 모집)
    
    타겟과 목적
    
    - 인플루언서 : 탑급 인플루언서를 제외한, 돈을 위해 1:1 채팅에 응할 수 있는 인플루언서
    - 후원자 : 내가 원하는 인플루언서를 후원하고 인플루언서와 1:1 채팅을 바라는 사람
    
    핵심 정책
    
    - 후원 → 인플루언서가 수락 → 10만원 이상 후원에대해 필수 채팅 3회 강제, 채팅안하면 후원금 못받아
    - 1:1 채팅 + 후원자 전용 피드를통해 사용자가 리워드 받아
    
    장점 : 
    
    - 인스타 dm 과 차이점 : 후원금이 걸리고 채팅에 강제성이 부여된 만큼 인플루언서와 더 확실한 1:1 채팅이 가능
    - 다른 리워드 대비 직관적이고 자극적
    
    단점 : 
    
    - 인플루언서의 부담감 : 부담감, 시간문제 등으로 1:1채팅안해서 후원금 수락 못하면 회사 손해
    - ‘채팅을 돈으로 산다’ 는 부정적 느낌
    
    서비스 화면
    
    - 인플루언서 화면 - 인스타그램 형태, / 후원 받은 메시지 : 금액별로 리스트업, 적은 금액의 후원금 - 포기하거나 1:1 채팅 필수 3회만 / 큰금액의 후원금 - 더 많은 채팅 하지않을까
    - 후원자 화면 → 인스타그램 형태, 내가 후원한 아티스트의 피드만떠 / dm 형식의 메시지 창
    
    ## < 확고한 타겟층 >
    
    1. 인기가 적고 돈을 많이 못버는 아이돌 → 더 큰 리워드를 받을 수 있음 (후원자에게 더 긴 채팅시간, 포스터 등 소비자가 지불한 금액에 충분히 대응할 것으로 예상)
    2. 인기가 있는 아이돌 → 리워드가 부족할 수 있음 (응원하는 수요는 많으나 아이돌이 1:1 채팅, 포스터 등에 큰 시간을 안쓸것 또는 아에 받지 않을 수도있음)
    
    ## < pain point >
    
    - 소비자 니즈
        
        아이돌에게 직접적으로 응원하고 1:1 채팅 하고싶어 → 타 플랫폼은 후원한다고 1:1 채팅 할 수 있는거 아니야
        
    - 창작자 니즈
        
        직관적인 수익 창줄 - 먼저 나와있는 수익 + 내가 선택 가능 → 바로 쌓이는것처럼 보이는 내 후원금액 (한달뒤 정산)
        
    
    ## < 핵심 매력포인트 >
    
    1. 아이돌이 등록안해도 내가 원하는 누구든지 웹상에 등록 가능
    2. 아이돌에게 수수료 10% 만 때고 온전히 후원금 전달
    3. 아이돌과 1:1 채팅, 2만원 이상 후원시 아티스트는 후원금 수락할려면 5회이상의 필수 채팅 필요 / 후원자의 환불은 불가
